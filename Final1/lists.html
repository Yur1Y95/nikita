Введение в списки

Списки могут содержать:
  • Неупорядоченную информацию (маркированный список - UL)
  • Упорядоченную информацию (нумерованный список - OL)
  • Определения (спискок определений - DL)

1. Маркированный список UL

Маркеры списка
  • Диск (type="disc")
  • Кружок (type="circle")
  • Квадрат (type="square")

2. Нумерованный список OL

Атрибуты списка:
  1. START - номер первого элемента
  2. TYPE - тип нумерации (1,a,A,i,I)
  3. reversed - обратная нумерация

3. Спискок определений DL

Список терминов
HTML
HyperText Markup Language - язык разметки
CSS
Cascading Style Sheets - таблицы стилей
XML
eXtensible Markup Language - расширяемый язык разметки
XSL
eXtensible Stylesheet Language - расширяемый язык описания стилей
